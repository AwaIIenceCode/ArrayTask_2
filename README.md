Even Numbers Task (JavaScript)
Task Description

Slightly more advanced basic task:

Create an array of 20 random numbers.

Create a new array that contains only even numbers from the first array.

Calculate the sum of even numbers.

Output how many even numbers were found.

Do not use .filter(). Use a regular loop instead.

Requirements

Use Math.random() to generate numbers.

Use a for loop to iterate through the array.

Use the modulo operator (%) to check if a number is even.

Manually push even numbers into a new array.

Keep track of:

Sum of even numbers

Count of even numbers

Expected Concepts Practiced

Arrays

Loops (for)

Conditionals (if)

Modulo operator (%)

Basic accumulation logic (sum & counter)
